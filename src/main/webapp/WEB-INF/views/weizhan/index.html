某某农业科技开发有限公司

某某农业科技开发有限公司始创于1992年，是一家集 科研、生产、销售、技术服务于一体的规模化专业型农业 科技公司。公司本着“科技为民所用”的原则，致力于农 业科技成果的转化和推广，在研发具有自主知识产权产品 的同时，以亚洲为中心，积极引进和筛选来自世界各地的 优质产品资源。目前，公司拥有子公司四个，经营着大田 作物种子、蔬菜种子和农化产品三块业务，在企业经营管 理上实行董事会领导下的总经理负责制。按照现代企业制 度的要求，坚持有利于科研、有利于生产、有利于销售的 “三个有利于”原则；在管理机构的设置上则坚持减少层 次、职责分明、提高效率的原则。
